# VIRAL ROAD ⚡

Plataforma de Inteligência Artificial para estrategistas de conteúdo e criadores digitais.

## Tecnologias
- **Frontend:** React 19 + Vite
- **IA:** Google Gemini 3 Flash
- **Backend:** Supabase (Auth/Database) + Netlify Functions (Serverless)
- **Estilização:** Tailwind CSS + Framer Motion

## Funcionalidades
- **Gerador de Blueprints:** Roteiros técnicos baseados em ganchos virais.
- **Road Performance:** Auditoria tática de perfis sociais em tempo real.
- **Acervo Cloud:** Armazenamento seguro de estratégias.
- **Hook Library:** Biblioteca de ganchos magnéticos testados pelo algoritmo.
